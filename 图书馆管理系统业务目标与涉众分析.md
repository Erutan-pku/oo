# 图书馆系统业务目标与涉众分析报告

### 业务目标
* 为图书馆用户提供自助信息查询与在线预约、在线续借服务，提高办事效率，方便客户，为客户提供更好的服务。
* 规范图书馆书籍与借阅情况管理，提高工作效率和管理效能。
* 对图书馆座位进行统一管理，方便用户在高峰期预约等位。
* 为管理员和用户之间的信息交流提供一个便捷的平台。

### 涉众分析报告
* 图书馆系统涉众概要

  | 编号    | 涉众名称    | 涉众说明                        | 期望                                       |
  | ----- | ------- | --------------------------- | ---------------------------------------- |
  | SH001 | 图书馆领导部门 | 图书馆领导部门是图书馆日常管理与运营的责任人。     | (1)新的系统可以规范化运作<br>(2)新系统与老方案之间可以无缝衔接，系统过度不影响图书馆的正常运营。 |
  | SH002 | 图书管理员   | 对图书馆的书籍、座位、借阅情况进行日常管理的运营人员。 | (1) 对用户借阅历史信息进行查询和管理。<br>(2)对用户历史占座信息和信用情况进行查询和管理。<br>(3) 对书籍借阅情况与历史进行查询和管理。<br>(4)对图书馆座位状态信息进行查询和管理。<br>(5)对用户账户进行创建、删除和重建操作。<br>(6)对书籍信息进行创建、删除或修改。<br>(7)对座位信息进行创建、删除或修改。<br>(8)给特定用户发送信息或警告。<br>(9)在整个系统中发布公告。<br>(9)对特定用户的日常操作权限进行锁定。<br>(10)系统应自动对用户等位超时的行为作出反馈，为超时者提出警告并进行过号处理。<br>(11)系统可以自动为用户发送图书即将到期的警报。(12)系统自动封禁连续3次等位超时的用户。 |
  | SH003 | 借阅者     | 图书馆图书的借阅者与图书馆座位的日常使用者       | (1)创建用户账户。<br>(2)修改账户信息。<br>(3)通过多种字段查询指定图书信息与图书借阅信息。<br>(4)在线预约与取消预约图书。<br>(5)在线续借。<br>(6)在线查询自己的借阅记录与归还时间。<br>(7)查询当前座位的使用情况、预约情况与等位情况。<br>(8)可以在线预约座位或取消座位预约。<br>(9)可以在线申请等位或取消等位申请 |
